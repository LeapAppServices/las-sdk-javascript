解压后的文件包括：

lc.js           未经压缩的完整的 JavaScript SDK
lc-mini.js      经过压缩的完整的 JavaScript SDK
lc-core.js      未经压缩的包含核心功能的 JavaScript SDK
lc-core-mini.js 压缩后的包含核心功能的 JavaScript SDK

lc.js 和 lc-core.js 的主要区别是后者不包括 BackBone.js 的一些支持，如 LC.View、 LC.Collection、LC.Router 等，相对地，文件会更小一些。
